DamsoViz Développement Web

DataViz est le projet de développement web de notre 3ème semestre de BUT MMI. Son objectif est de créer une page web qui permettra de visualiser des données à propos d'un phénomène qui évolue. Dans notre cas, nous avons choisi comme thème le rappeur Belge Damso et avons décidé d'afficher des données sur l'évolution de ses statistiques au fil du temps.

Compétences utilisées

HTML / CSS / JavaScript / Chart.js / AOS
